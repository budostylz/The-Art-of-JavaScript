## Writing a Test
https://youtu.be/7R4mauxWXMk

```javascript

expect(add(0.1, 0.2)).toBe(0.3);


expect(// starting point of test
    
    add(//'actual' of what to test
        0.1,
        0.2
    )
)

.toBe(//comparison or matcher of what to expect out of the test
    0.3
   );

//after evaluation
add(0.1, 0.2) === 0.3

```
